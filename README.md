# API - Laravel 5.6

Cara Install

-   Buat Database
-   Masuk ke project api
-   Rename file .env.example => .env
-   Setting database di .env
-   php artisan key:generate
-   composer install
-   php artisan migrate
-   php artisan passport:install
-   php artisan serve
-   List item
